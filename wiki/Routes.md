# Маршруты

## Как настраивать маршрут через GUI

Маршрут настраивается из `setup GUI`:

1. Откройте караван.
2. Нажмите `Route`.
3. В меню маршрута используйте:
   - `Add Route Stop`
   - `Start Route`
   - `Stop Route`
   - `Clear Route`
   - переключатель loop mode

## Как добавить точку маршрута

1. Откройте `Route`.
2. Нажмите `Add Route Stop`.
3. Выберите город из списка.
4. Плагин выберет случайный `Shop Plot`.
5. Введите в чат время стоянки в минутах.

После этого остановка сохранится в маршрут каравана.

## Как выбираются доступные Towny города

Плагин фильтрует города по:

- `allow-own-town`
- `allow-allied-towns`
- `allow-neutral-towns`
- `allow-enemy-towns`

Также исключаются:

- невалидные/удалённые города;
- города без `Shop Plot`.

Если Towny не может надёжно определить отношение, плагин старается не пускать только явно вражеские города.

## Как выбирается random Shop Plot

После выбора города:

1. плагин собирает список всех `Shop Plot`;
2. берёт один случайный участок;
3. вычисляет точку рядом с центром;
4. если чанк уже загружен, корректирует Y по поверхности;
5. если нет, использует безопасный fallback.

Плагин не force-load’ит чанки ради этого.

## Как задаётся время стоянки через чат

После выбора города игрок получает запрос в чат:

- ввести длительность остановки в минутах;
- или написать `cancel`.

Значение проверяется по:

- `route.min-stop-minutes`
- `route.max-stop-minutes`

## Как работает route loop

Если loop mode включён:

- после последней остановки караван не идёт домой;
- он снова начинает маршрут с первой остановки.

Если loop mode выключен:

- после последней остановки караван запускает возврат домой.

## Как работает возврат домой

Домашняя точка задаётся автоматически при первом успешном физическом спавне каравана.

Команды возврата:

```text
/caravan return <id>
/caravan admin return <id>
```

После завершения обычного маршрута без loop mode караван также возвращается домой автоматически.

## Как работает ETA

ETA рассчитывается по прямой:

- расстояние до цели;
- текущая скорость `speed_blocks_per_second`.

ETA:

- уменьшается по мере движения;
- не уходит в отрицательные значения;
- становится `0` при прибытии.
