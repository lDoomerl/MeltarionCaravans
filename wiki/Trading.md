# Торговля

## SELL-операции

SELL-операция означает:

- караван продаёт товар из своего инвентаря;
- операция привязана к конкретному резервному слоту;
- товар должен физически лежать в этом слоте.

При покупке игроком плагин проверяет:

1. операция активна;
2. караван на Shop Plot;
3. караван `STOPPED`;
4. слот всё ещё содержит нужный предмет;
5. количество в слоте достаточно;
6. у покупателя есть валюта;
7. у покупателя есть место;
8. у каравана есть место для принятия валюты.

## BUY-операции

BUY-операция означает:

- караван хочет купить у игроков определённый предмет;
- указывается количество за одну сделку;
- цена;
- максимум общего объёма.

При продаже игроком в BUY-заявку проверяется:

1. операция активна;
2. караван на Shop Plot;
3. караван `STOPPED`;
4. объём заявки ещё не исчерпан;
5. у игрока есть нужный предмет;
6. у каравана есть место для приёма товара;
7. у каравана есть валюта для оплаты;
8. у игрока есть место для получения валюты.

## Валюта предметом

На текущем этапе валюта — это **предмет**, а не баланс через экономический API.

Настраивается в:

```yml
currency-item: GOLD_INGOT
```

## Как работает антидюп

Вся торговая логика строится так, чтобы не было частично завершённых обменов:

- заранее проверяется место в обоих инвентарях;
- заранее проверяется наличие валюты и предметов;
- используются snapshots инвентарей;
- при ошибке позднего шага выполняется rollback;
- торговля блокируется lock’ом по caravan id.

## Что проверяется перед транзакцией

Перед реальной выдачей/списанием предметов плагин проверяет:

- валидность операции;
- принадлежность операции каравану;
- активность операции;
- текущий статус каравана;
- наличие Shop Plot;
- содержимое слота или наличие similar items;
- пространство в инвентарях;
- валюту;
- состояние заявки/остатка.

## Почему торговля доступна только на Shop Plot и только когда караван STOPPED

Это сделано специально для ограничения геймплея:

- торговля должна быть привязана к торговым точкам Towny;
- движущийся караван не должен выступать мобильным магазином в любом месте;
- статус `STOPPED` делает торговлю понятной и предсказуемой.
