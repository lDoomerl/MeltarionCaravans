## Что такое MeltarionCaravans

`MeltarionCaravans` — это плагин для Paper/Leaf 1.21.x, который добавляет караваны как постоянные игровые объекты с:

- собственным хранилищем;
- публичной торговлей через Towny Shop Plot;
- виртуальным перемещением;
- физической проекцией в мире;
- маршрутами с несколькими остановками;
- GUI-настройкой для владельца и администратора.

## Кратко о механиках

- Караван создаётся не бесплатно: для него нужна **лицензия**.
- У каждого каравана есть **постоянный инвентарь** и **торговые операции**.
- Караван движется в первую очередь **виртуально**.
- Физические сущности `Wandering Trader + 2 Trader Llamas` появляются только когда рядом есть игроки.
- Публичная торговля доступна только:
  - на **Towny Shop Plot**;
  - когда караван находится в статусе `STOPPED`.
- Маршрут настраивается через GUI и может включать несколько остановок.
- Для внешних плагинов доступен **read-only Bukkit API** через `ServicesManager`.

## Требования

- Java `21`
- Paper или Leaf `1.21.x`
- Towny
- Dynmap — опционально

## Быстрый старт

1. Скачайте jar плагина из Releases.
2. Поместите jar в папку `plugins/`.
3. Запустите сервер.
4. Проверьте, что созданы:
   - `config.yml`
   - `lang.yml`
   - `gui.yml`
5. Выдайте себе лицензию:

```text
/caravan admin givelicense <ник> 1
```

6. Создайте караван:

```text
/caravan create
```

7. Призовите его в городе Towny:

```text
/caravan spawn 1
```
